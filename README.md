# [Pullwave](http://www.pullwave.com) 的 简单可视化 Python 实现

## 依赖
* matplotlib

## 用法
直接运行程序，会提示输入 词语，然后会绘制柱状图。

目前 2017-05-15 还只能检索一个词，两个词的还需要等一阵子，并且还比较糙，细节上还有很多要改进的。
